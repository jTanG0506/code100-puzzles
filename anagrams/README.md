## Anagrams

You will receive an array of strings.

Submit that array of strings, sorted from A-Z, but only supply the words that are an anagram of another word in that array.

### Example puzzle

```
["kiwi", "melon", "apple", "lemon"]
```

### Example solution

```
["lemon", "melon"]
```
